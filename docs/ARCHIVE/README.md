Archive rules:

1. Move docs here only after checking active links and source-of-truth status.
2. Do not delete archived docs until their unique content has been merged or declared stale.
3. If an archived doc becomes active again, move it back and update README/index links.
4. Do not treat historical examples or old standards in this folder as current policy.
