# Security Review Workflow

## Use This Prompt

For agent execution, use `../prompts/review-security.md`.

This guide explains the review standard. The prompt is the runnable checklist.

Use for auth, authorization, sensitive data, file handling, command execution, dependencies, configuration, migrations, and external integrations.

Output findings by severity and include evidence, risk, recommendation, and required human decision.

Do not modify code unless explicitly asked.
