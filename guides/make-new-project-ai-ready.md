# Make a New Project AI-Ready

Use when starting a new repository.

## Steps

1. Inspect stack and project structure.
2. Create `AGENTS.md`.
3. Create project docs for context, architecture, development, testing, and security.
4. Add PR and issue templates.
5. Add `scripts/verify.sh` if practical.
6. Mark unknown values as `Needs confirmation`.
7. Run validation.

Expected output: a project where OpenCode can read local instructions before acting.
