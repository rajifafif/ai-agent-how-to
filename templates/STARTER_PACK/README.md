# Starter Pack

Copy/adapt this minimum set into a real project first. Do not copy the whole toolkit.

## Minimum Project Files

```text
AGENTS.md
docs/PROJECT_CONTEXT.md
docs/ARCHITECTURE.md
docs/DEVELOPMENT.md
docs/TESTING.md
docs/SECURITY.md
.github/pull_request_template.md
scripts/verify.sh
```

## Suggested Copy Order

1. `templates/AGENTS.template.md` -> project `AGENTS.md`.
2. Project context templates -> project `docs/`.
3. PR/checklist templates -> project `.github/` or `templates/`.
4. Install skills in the AI agent/harness if supported.
5. Fill real commands and mark unknowns as `Needs confirmation`.
6. Run project validation.

## Do Not

- Do not overwrite existing project docs blindly.
- Do not invent commands.
- Do not copy all maturity modules at once.
